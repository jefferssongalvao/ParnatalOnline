Será disponibilizado um arquivo denominado "TrabalhoEDBLP.zip"
que contem vários arquivos, entre eles a pasta do projeto no Netbeans
(com os fontes). Assim para gerar o executável siga as intruções a se-
guir:

1 - Extraia o conteúdo do arquivo "TrabalhoEDBLP.zip" em uma
pasta de sua preferência, ex. : "C:/".

2 - Navegue até a pasta que você escolheu no passo "1". Ao chegar lá
copie a pasta "Parnatalandia" para a sua pasta "workspace" do eclipse.

3 - Abra o Eclipse e escolha a seguinte opção na barra de menus,
File > Import..., na janela que abrir escolha a opção:
General > Existing Projects into Workspace, clique em Next.
Na opção "Select root directory" escolha a sua pasta de
workspace. Selecione o projeto "Parnatalândia" e clique em
Finish.

4 - Escolha a seguinte opção na barra de menus no Eclipse:
File > Export > Java (JAR file) > Select the export destination > 
Next > Finish.

5 - Navegue até a pasta que você escolheu como pasta destino no
passo anterior. Ao chegar lá, você encontrará o executável ".jar".
